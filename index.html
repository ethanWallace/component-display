Component-display The component-display component displays a component's attributes, slots, and events in a tabbed interface. Features
  • Code frame: Displays the live component and the mark up to render the component in different frameworks
  • Attributes: Displays component attributes in a tabbed interface
  • Slots: Displays component slots in a tabbed interface
  • Events: Displays component events in a tabbed interface
  • Accessibility: Allows running an accessibility test against the live component
